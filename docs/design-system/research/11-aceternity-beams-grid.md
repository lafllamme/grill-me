## Observed system

- Sparse diagonal beams and a faint grid give an otherwise neutral hero direction and depth.
- The main product viewport is a large rounded shell with a soft outer frame.
- Background detail is masked near text and stronger toward the edges.

## Grillme translation

- Use Prism as the richer equivalent of the beam field.
- Mask or darken the animation directly behind long copy.
- Let brighter red trajectories cross empty space and stage edges, not body text.
- A subtle grid may appear inside the roast stage to suggest analysis, but should not become a global cyberpunk motif.

## Behavior and extractable components

- Background contrast is reduced directly behind copy and allowed to peak in empty margins.
- Extract this masking rule for Prism: readable text zones stay dark while brighter trajectories cross gutters and panel edges.
- Any grid belongs only inside the agent-analysis viewport.
